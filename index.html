NetBox Community Helm Charts

This is the repository for the NetBox community Helm charts, version 5.0 and up. It hosts the indexes that define the versions and resources needed to install NetBox inside a Kubernetes cluster.

For source code and details, see the GitHub repository.

For the previous 4.x charts maintained by Chris Boot, please use the old repo at https://charts.boo.tc/.

The indexes are auto-generated and are not meant to be human-readable, but you can find the current helm chart index here.
